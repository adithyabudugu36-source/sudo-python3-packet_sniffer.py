# 📡 Packet Sniffer (Python)

A simple packet sniffer built using Python and Scapy for educational cybersecurity learning.

## 🚀 Features
- Captures live network packets
- Displays Source IP, Destination IP, and Protocol
- Lightweight command-line tool

## 🛠 Requirements
- Python 3
- Scapy library

Install Scapy:
pip install scapy

## ▶️ How to Run
sudo python3 packet_sniffer.py

## ⚠️ Disclaimer
For educational purposes only. Use only on networks you own or have permission to test.
